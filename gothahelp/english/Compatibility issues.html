Compatibility issues

OpenGotha supports tournament data generated by other programs.

Importing players and games from Geoff Kaniuk's GoDraw program

From GoDraw 6.3.6, use "File .. Export .. Ratings" menu item.

Save the file into a .h9 file

In OpenGotha, create a new tournament.
Then import the .h9 file with the "Tournament .. Import .. Import Players and Games from h9 file" menu item
